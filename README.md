# Sigil
Sigil is a rogue like game about a wizard named Sigil who is lost in a forest and needs to complete quests to get back to a city.

The main point of this project is that the entire game logic is written inside of one main Lambda meaning that this entire game is written on one line. Why? Because I wanted to try to push the limits of Python and I feel like I was successful at that.
